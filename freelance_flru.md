## Услуга 2: Telegram AI Bot (RAG + Admin)

### Название
Разработка Telegram-бота с AI (RAG-система, база знаний, админ-панель)

### Категория
Программирование → Разработка Чат-ботов

### Изображения (загрузить)
- `telegram_bot_cover.png` — обложка
- `telegram_bot_selling_1.png`
- `telegram_bot_selling_2.png`
- `telegram_bot_selling_3.png`

### Цена / Срок
от 30 000 ₽ / 3-5 дней

### Результат для заказчика (формат передачи)
```
• Готовый работающий Telegram-бот (токен + ссылка)
• Репозиторий или ZIP: .py, requirements.txt, .env.example, docker-compose.yml
• Админ-панель Streamlit (запускается отдельно)
• Документация по установке и использованию
• Инструкция по загрузке документов
• Видео-демонстрация работы
```

### Описание (HTML — поле "Описание")
```
Создам Telegram-бота с искусственным интеллектом, который отвечает на вопросы ваших клиентов 24/7 на основе ваших документов.

Как работает:
1. Вы загружаете документы (PDF, DOCX, TXT, Excel)
2. Клиент пишет боту вопрос в Telegram
3. Бот мгновенно находит ответ в вашей базе знаний
4. Каждый ответ содержит источник (файл + цитата)

Функционал:
• RAG-система — Mistral Embeddings + ChromaDB
• FAQ — быстрые ответы на частые вопросы
• Админ-панель Streamlit — управление документами, FAQ, аналитика
• Кнопка "Полезно?" — сбор обратной связи
• Статистика — количество вопросов, точность, популярные темы

Для кого:
• Интернет-магазины — вопросы о товарах, доставке
• Онлайн-школы — поддержка учеников
• Компании — внутренняя база знаний
• Медицинские/юридические фирмы — ответы по документам

Стек: Python, FastAPI, Telegram Bot API, ChromaDB, Mistral AI (бесплатно), Streamlit, Docker
```

### Заказчик получит (HTML — поле "Заказчик получит")
```
• Telegram-бот с RAG-системой
• Админ-панель Streamlit (управление документами, FAQ, аналитика)
• Загрузка до 50 PDF-документов
• Кнопка "Полезно?" + сбор статистики
• Docker-Compose, документация, исходный код
• 1 неделя поддержки после сдачи
• 2 бесплатные доработки
```

### Что нужно от заказчика (HTML — поле "До начала работы заказчик должен предоставить")
```
• Документы (от 5 PDF, DOCX, TXT)
• Токен Telegram-бота (@BotFather, бесплатно)
• Примеры типовых вопросов клиентов
• Сервер для деплоя (или ваш ПК для теста)
```

### Теги / Навыки
Python, Telegram Bot, AI, RAG, Mistral, Chatbot, FastAPI, Streamlit, Docker, ChromaDB, LangChain
