RAG Corp Bot

Lightweight corporate document Q&A for Telegram

✅ Ready for Deployment

How It Works

Upload any PDF to the Telegram bot. It parses the document, splits it into semantic chunks, embeds them with Mistral AI, and stores in ChromaDB. Employees can then ask natural-language questions and get instant answers grounded in your documents.

User uploads PDF ──▶ Bot chunks + embeds ──▶ ChromaDB │ User asks question ──▶ Semantic search ──▶ Mistral generates answer ──▶ Reply
